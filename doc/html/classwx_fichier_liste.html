Recherche dans le fichier si la combinaison le numéro de ligne de chaineT.

position spécifie à partir de quel endroit rechercher. Si position vaut -1, *la recherche se fait depuis le début du fichier. La recherche est effectuée en premier en sens descendant puis en sens montant à partir de position

positionl'endroit à partir duquel doit être fait la recherche
